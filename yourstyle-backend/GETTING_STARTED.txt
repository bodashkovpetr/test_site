🚀 QUICK START (Windows):

1. Double-click: QUICKSTART.bat
   (This automates steps 2-6 below)

   OR manually:

2. Install dependencies:
   npm install

3. Configure environment:
   copy .env.example .env
   Edit .env with your passwords

4. Start Docker:
   docker-compose up -d

5. Setup database:
   npm run migrate
   npm run seed

6. Start server:
   npm run dev

7. Test it works:
   Open: http://localhost:3000/health
